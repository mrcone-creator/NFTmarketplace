# 📖 ユーザーマニュアル — NFT Marketplace

> **対象**: マーケットプレイスを利用するユーザー向けガイド

---

## 🚀 はじめに

NFT Marketplace は、運営が厳選したプレミアム NFT を購入・コレクションできるマーケットです。  
MetaMask などの Web3 ウォレットを接続することで、簡単に NFT を購入できます。

---

## 1️⃣ ウォレットの接続

### 対応ウォレット
| ウォレット | 対応OS | ダウンロード |
|-----------|--------|------------|
| MetaMask | Chrome/Firefox/Edge | [metamask.io](https://metamask.io) |
| WalletConnect | モバイル | AppStore / Google Play |
| Coinbase Wallet | Chrome/モバイル | [coinbase.com/wallet](https://www.coinbase.com/wallet) |

### 接続手順
1. 画面右上の **「ウォレット接続」** ボタンをクリック
2. 使用するウォレットを選択
3. ウォレットアプリで接続を承認
4. ✅ アドレスが表示されれば接続完了

> ⚠️ **MetaMask がインストールされていない場合**はデモモードで接続されます

---

## 2️⃣ NFT の探し方

### カテゴリフィルター
- **すべて** — 全カテゴリを表示
- **アート** — デジタルアート作品
- **コレクタブル** — 限定コレクターズアイテム
- **ミュージック** — 音楽 NFT
- **ゲーム** — ゲームアイテム・キャラクター

### 検索
- 検索バーに NFT 名・説明のキーワードを入力

### 並び替え
| 並び替え | 説明 |
|---------|------|
| 最新順 | 最近発行された NFT を先に表示 |
| 価格: 安い順 | 低価格 NFT を先に表示 |
| 価格: 高い順 | 高価格 NFT を先に表示 |
| 人気順 | お気に入り数の多い NFT を先に表示 |

---

## 3️⃣ NFT の購入方法

1. 購入したい NFT カードをクリック
2. 詳細モーダルで内容を確認
   - 名前・説明・価格 (ETH/円換算)
   - レアリティ・プロパティ
3. **「💎 今すぐ購入」** ボタンをクリック
4. ウォレットで取引を承認
5. 🎉 購入完了！NFT があなたのコレクションに追加されます

> 💡 **ウォレット未接続の場合**はウォレット接続画面が表示されます

---

## 4️⃣ マイコレクション

ナビゲーションの **「マイコレクション」** から確認できます。

- 購入した全 NFT の一覧を表示
- NFT をクリックして詳細を確認
- ウォレット未接続の場合は接続が必要

---

## 5️⃣ アクティビティ

ナビゲーションの **「アクティビティ」** から確認できます。

- マーケット全体の最近の取引履歴を表示
- 購入・出品・なりなど各種アクションを確認

---

## 6️⃣ お気に入り

NFT カードの ❤️ ボタンをクリックしてお気に入り登録できます。
- お気に入りは **「人気順」** の並び替えに影響します

---

## ❓ よくある質問

**Q: ウォレットを接続しないと購入できますか？**  
A: いいえ、購入にはウォレット接続が必要です。

**Q: どのネットワーク (チェーン) に対応していますか？**  
A: 現在 Ethereum を主として対応しています。

**Q: 購入したNFTはウォレットに届きますか？**  
A: デモ環境では、購入データはブラウザのローカルストレージに保存されます。本番環境では実際のブロックチェーンに記録されます。

**Q: ウォレットを切断するには？**  
A: 画面右上のウォレットアドレスをクリックすると切断ボタンが表示されます。
