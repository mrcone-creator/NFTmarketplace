## 🚀 はじめに

NFT Marketplace は、**Silver / Gold / Platina** の3種類のプレミアム NFT を購入できるマーケットです。  
NFT を保有し、ステーキングすることで **四半期ごとに USDT 報酬** を受け取れます。

### 主な特徴

| 機能 | 内容 |
|------|------|
| 🛍️ NFT購入 | 3種類のプレミアムNFTを購入 |
| 🔒 ステーキング | 保有NFTを運用してUSDT報酬を獲得 |
| 💰 USDT報酬 | 四半期ごとに購入価格に応じた報酬を受取 |
| 🦊 ウォレット対応 | MetaMask / WalletConnect / Coinbase Wallet |

---

## 1️⃣ ウォレットの接続

### 対応ウォレット

| ウォレット | 対応環境 | ダウンロード |
|-----------|---------|------------|
| MetaMask | Chrome / Firefox / Edge | [metamask.io](https://metamask.io) |
| WalletConnect | モバイル対応 | AppStore / Google Play |
| Coinbase Wallet | Chrome / モバイル | [coinbase.com/wallet](https://www.coinbase.com/wallet) |

### 接続手順

1. 画面右上の **「ウォレット接続」** ボタンをクリック
2. 使用するウォレットを選択
3. ウォレットアプリで接続を承認
4. ✅ アドレスが右上に表示されれば接続完了（例: `0x1234...abcd`）

> ⚠️ **MetaMask がインストールされていない場合**はデモモードで接続されます（シミュレーション）

### ウォレットの切断

右上のウォレットアドレスをクリック → トースト通知の **「切断」** ボタンをクリック

マーケットには以下の3種類のNFTが用意されています。

| 項目 | 内容 |

### 🥇 Gold NFT

|------|------|

| 四半期報酬 | **500 USDT** |
| 年間報酬 | **2,000 USDT** |

### 💎 Platina NFT

| 項目 | 内容 |
|------|------|
| 価格 | ¥2,000,000（約 5.0 ETH） |
| APR | 36% |
| 発行枚数 | 最大50枚（管理者が変更可能） |
| 四半期報酬 | **1,200 USDT** |
| 年間報酬 | **4,800 USDT** |
| 特徴 | 最高利回りのプレミアムプラン（50枚限定） |

### 報酬計算式

```
四半期 USDT 報酬 = 購入価格(¥) × APR ÷ 4 ÷ 150(¥/USDT)
```

> デモ環境レート: 1 ETH = ¥400,000 / 1 USDT = ¥150

## 3️⃣ NFT の購入方法

### 購入手順

1. 上部ナビゲーションの **「マーケット」** をクリック
2. 購入したいNFT（Silver / Gold / Platina）の **「💳 購入する」** ボタンをクリック

   > ⚠️ ウォレット未接続の場合はウォレット接続画面が先に表示されます

3. **購入確認モーダル** で内容を確認
   - NFT名 / APR / 購入価格（¥ / ETH / USDT換算）
   - 四半期報酬額 / 年間報酬額
4. **「購入する」** ボタンをクリック
5. ⛓️ ブロックチェーン処理（約3秒のシミュレーション）
6. 🎉 購入完了！NFTがウォレットに届きます

### 残枚数について

- 各NFTに残枚数がリアルタイム表示されます
- 売り切れの場合はボタンが **「🚫 SOLD OUT」** に変わり購入不可になります

ナビゲーションの **「マイコレクション」** から確認できます。

## 5️⃣ ステーキング

ナビゲーションの **「💰 ステーキング」** から操作できます。

### ステーキングとは

NFTを「ステーキング」することで USDT 報酬の受け取りが有効になります。  
報酬は管理者が四半期ごとに手動でウォレットアドレスに配布します。

### ステーキングの手順

1. **「💰 ステーキング」** ページへ移動
2. 所有しているNFTの一覧が表示されます
3. ステーキングしたいNFTの **「🔒 ステーキング開始」** ボタンをクリック
4. ✅ 「運用中」バッジが表示されれば有効化完了

### ステーキングページの統計

| 統計 | 説明 |
|------|------|
| 保有NFT数 | 購入済みの全NFT数 |
| 運用中NFT | ステーキング中のNFT数 |
| 次回配当合計 | 次回の四半期で受け取るUSDT合計 |
| 年間報酬合計 | 全保有NFTの年間USDT報酬合計 |

### 報酬スケジュール

| タイミング | 内容 |
|-----------|------|
| 購入日から3ヶ月後 | 第1回 報酬配布日 |
| 購入日から6ヶ月後 | 第2回 報酬配布日 |
| 購入日から9ヶ月後 | 第3回 報酬配布日 |
| 購入日から12ヶ月後 | 第4回 報酬配布日 |

報酬配布は **管理者が手動で実行** します。  
配布後、ステーキングページの「配布済み報酬」欄に記録が表示されます。

## ❓ よくある質問

**Q: ウォレットを接続しないと購入できますか？**  
A: いいえ、購入にはウォレット接続が必要です。

**Q: MetaMaskを持っていない場合は？**  
A: WalletConnect や Coinbase Wallet でも接続できます。MetaMask未インストールの場合はデモモードで動作します。

**Q: 購入したNFTはいつから報酬が発生しますか？**  
A: ステーキングを開始してから四半期経過後（3ヶ月後）の報酬発生日に管理者から配布されます。

**Q: 報酬はいつ受け取れますか？**  
A: 報酬発生日に管理者が手動で配布を実行します。ステーキングページの次回報酬日をご確認ください。

**Q: ステーキングを解除できますか？**  
A: 現バージョンではステーキング後の解除には非対応です。

**Q: 購入価格や報酬の換算レートは？**  
A: デモ環境では 1 ETH = ¥400,000 / 1 USDT = ¥150 を使用しています。実際のレートとは異なります。

**Q: ウォレットを切断するには？**  
A: 右上のウォレットアドレスをクリックするとトースト通知に「切断」ボタンが表示されます。
