# 👑 管理者マニュアル — NFT Marketplace Admin Panel

> **対象**: マーケットプレイスの運営管理者向けガイド  
> **アクセス**: `admin/index.html` からアクセス

---

## 🚀 管理者パネルへのアクセス

```
http://localhost:[ポート]/admin/index.html
ファイルパス: NFTmarketplace/admin/index.html
```

---

## 📊 ダッシュボード

管理者パネルにアクセスすると最初に表示されます。

### 統計カード
| カード | 説明 |
|-------|------|
| 総NFT数 | 管理者が発行した全NFT数 |
| 総売上 (ETH) | 累計売上高 (ETH) |
| 公開中NFT | 現在マーケットに公開中のNFT数 |
| コレクター数 | NFTを購入したユーザー数 |

### 週次売上チャート
- 直近7日間の日別売上推移をバーチャートで表示

### 最近のアクティビティ
- 発行・販売・公開などの最新5件を表示

### 最近発行したNFT
- 最新5件のNFTを一覧表示・クイック操作が可能

## ✨ NFT の発行 (ミント)

サイドバーの **「NFT発行」** または右上の **「✨ NFT発行」** ボタンから。

### 入力項目

| 項目 | 必須 | 説明 |
|-----|------|------|
| NFT名 | ✅ | NFTの名前 (最大80文字) |
| 絵文字アイコン | ✅ | NFTを代表する絵文字 (50種類から選択) |
| 背景グラデーション | ✅ | カードの背景色 (10種類から選択) |
| カテゴリ | ✅ | アート / コレクタブル / ミュージック / ゲーミング |
| レアリティ | ✅ | Common / Uncommon / Rare / Epic / Legendary / Mythic |
| 価格 (ETH) | ✅ | 販売価格 (0.001 ETH 以上) |
| エディション | — | 例: `#001 / 100` |
| 説明 | ✅ | NFTの説明文 |
| 追加プロパティ | — | カスタムkey-valueプロパティ |
| Featured設定 | — | ヒーローセクションに大きく表示 |
| 公開ステータス | ✅ | 公開 / 下書き |

### 発行手順
1. 各項目を入力
2. **プレビュー** でNFTの見た目を確認
3. **「✨ NFTを発行する」** をクリック
4. 約2秒のシミュレーション後に発行完了
5. 自動的にNFT管理ページへ移動

## 🎨 NFT の管理

サイドバーの **「NFT管理」** から。

### 一覧表示
- 全発行NFTを表形式で表示
- 検索バー・ステータスフィルターで絞り込み可能

### 操作ボタン
| ボタン | 説明 |
|-------|------|
| ✏️ (編集) | NFT名・価格・カテゴリ・説明・ステータスを編集 |
| ✅ (公開) | 下書きNFTを公開する |
| 📝 (下書き) | 公開中NFTを非公開にする |
| 🗑️ (削除) | NFTを完全に削除 (確認ダイアログあり) |

### ステータス
| ステータス | 説明 |
|----------|------|
| 🟢 公開中 | マーケットプレイスに表示・購入可能 |
| 🟡 下書き | 管理者のみ確認可能・非公開 |
| ⚫ アーカイブ | 非表示・保存のみ |

## 💰 売上管理

サイドバーの **「売上管理」** から。

### 表示情報
- **総売上 ETH**: 全取引の合計額
- **総売上 (万円)**: ETHレート換算の日本円額
- **販売件数**: 完了した取引数
- **平均単価**: NFT1件あたりの平均価格

### 取引履歴テーブル
| 列 | 説明 |
|---|------|
| NFT | 購入されたNFT名・絵文字 |
| 購入者 | ウォレットアドレス (短縮表示) |
| 価格 (ETH) | 取引額 |
| 円換算 | 日本円換算額 |
| 取引日時 | 月/日 時:分 |
| TX Hash | トランザクションハッシュ (一部) |

---

## 📋 アクティビティ

サイドバーの **「アクティビティ」** から。

- 管理者操作の全ログを時系列で表示
- NFT発行・編集・公開・削除など全操作を記録
- **「ログをクリア」** ボタンで全ログを削除可能

---

## ⚙️ 設定

サイドバーの **「設定」** から。

### マーケット設定
| 設定 | 説明 |
|------|------|
| マーケットプレイスを公開 | ONでユーザーにマーケットを公開 |
| 新規購入を許可 | OFFで購入ボタンを無効化 |
| おすすめ表示 | ヒーローセクションのFeatured NFT表示 |

### レート設定
- **1 ETH = 円**: 表示用のETH/JPY換算レート
- デフォルト: ¥400,000

### データ管理
| 操作 | 説明 |
|------|------|
| 全NFTデータをリセット | 管理者発行NFTを全削除 (デフォルトNFTは残る) |
| 購入履歴をクリア | ユーザーの購入・所有データを全クリア |

## 📝 データについて

全データはブラウザの **localStorage** に保存されます。

| ストレージキー | 内容 |
|-------------|------|
| `nftm_admin_nfts` | 発行したNFTデータ |
| `nftm_settings` | 管理者設定 |
| `nftm_admin_activity` | 管理者アクティビティログ |

> ⚠️ ブラウザのキャッシュをクリアするとデータが消えます。  
> 本番環境ではデータベース (Firebase / Supabase 等) の使用を推奨します。

## 🔒 セキュリティに関して

現在のバージョンでは認証機能は実装されていません。  
本番環境では以下を追加してください：

1. **管理者認証** (JWT / OAuth)
2. **アクセス制限** (IP制限 / 2要素認証)
3. **スマートコントラクト連携** (ERC-721 / ERC-1155)
4. **バックエンドAPI** (Node.js / Express)
5. **データベース** (PostgreSQL / MongoDB)
